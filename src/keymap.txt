模板快捷键：
1.sout 打印语句
2.main 主函数
3.fori/itar 普通for循环
4.iter 增强for循环

F2 定位高亮错误或警告位置

Ctrl + …
1.Y 删除当前行
2.D 复制当前行到下行
3.W 选中文本
4.backspace 删除整个单词
5.+/- 代码展开/折叠
6./ 单行注释
7.shift+/ 多行注释
8.F 查找文本
9.R 替换文本
10.O 重写
11.[/] 代码块开始/结束位置

Alt + …
1.enter 提示修复
2.insert 插入方法
3.left / right 切换代码视图
4.Up / Down 方法间快速移动定位
5.1 打开或隐藏 Project 面板
6./ 快速补全代码

Ctrl + Alt + …
1.L 格式化
2.T 增加包装

Ctrl + Shift + …
1.del 去掉包装
2.F10 运行
